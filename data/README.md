# Échantillon des données issues du pilote automatique


Le pilote émet des données de différents types et
à une cadence significative. 
Le logiciel NauteffVision les lit,
leur ajoute l'heure de lecture et les enregistre.
L'heure est celle du raspberry pi.

Ces enregistrement servent à comprendre
le fonctionnement du pilote et
à lui apporter des améliorations.

L'important volume de données issues du pilote est trop important
pour être stocké sur ce dépôt. Il n'y a qu'un petit échantillon.

Le répertoire inter contient un petit échantillon
d'une dizaine de minutes de fonctionnement.
